# course-search-skill

GEO 课程微信小程序的最小搜索能力。

## 目标

- 帮用户按课程名、讲师名、课程简介、课程标签搜索课程。
- 搜索命中多门课时，把用户接力到课程列表页，并预填搜索词。
- 搜索仅命中一门课时，把用户直接接力到课程详情页。

## 可用原子接口

- `searchCourses`

## 业务约束

- 只能使用用户原话中的搜索词，不要补造课程名。
- 当用户表达“随便看看课程”“推荐几门课”这类泛搜索意图时，可以留空 `keyword`，返回课程列表页。
- 搜索不到时，不要假装命中；要明确说明未找到，并建议用户换关键词。
- 当前 skill 只负责搜索与导览，不负责购买、兑换、客服、学习进度等后续动作。

## 输出约定

- 命中单门课程：一句话告知已定位课程，并引导用户点击卡片进入详情。
- 命中多门课程：一句话告知找到若干课程，并引导用户点击卡片查看列表。
- 未命中：一句话说明未找到，并引导用户点击卡片继续浏览全部课程。
